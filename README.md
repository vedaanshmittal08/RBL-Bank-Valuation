# RBL Bank Valuation – Dividend Discount Model (Standalone)

An independent Dividend Discount Model (DDM) valuation of RBL Bank developed using publicly available financial statements and annual reports.

---

## Overview

This repository contains an end-to-end standalone valuation of RBL Bank built in Microsoft Excel. The model forecasts the bank's financial performance through FY28E and estimates intrinsic value using a multi-stage Dividend Discount Model (DDM).

The project was developed independently to strengthen my understanding of bank valuation, financial modelling, forecasting, and equity research.

---

## Key Results

| Metric | Value |
|--------|------:|
| Intrinsic Value | ₹84/share |
| Market Price (March 2026) | ₹292/share |
| Implied Downside | 71% |

---

## Methodology

The valuation follows the following framework:

1. Historical financial statement analysis
2. Forecasting of key banking drivers
3. Projection of future dividends
4. Cost of Equity estimation using CAPM
5. Terminal Value estimation using the Gordon Growth Model
6. Sensitivity analysis across Cost of Equity and Terminal Growth assumptions
